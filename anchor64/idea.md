# Anchor
The best way to play N64 PC ports. Automatically download and update, create save data backups, manage individual save files, edit save data, create multiple instances (with different saves and settings), easily configure your controller, and start with the best settings for the ultimate experience.

This program would use the PC ports created by Harbour Masters.

## Ocarina of Time
This would use _Ship of Harkinian_ and would automatically adjust the settings (asking the player about a few options) to get the best improvements. It would include the OoT3D 4K texture pack (used with Djipi's 3DS Experience) and would have multiple HUDs to choose from.

### Unlockable Games
> [!NOTE]
> All ROM hacks would be slightly modified to work with the OoT4K models and textures.
After finishing the game, you would unlock the option to play the _Master Quest_ version of the game. (You could also unlock this at any time by choosing an option in settings.) You would also unlock the ability to play _Dawn & Dusk_, a homebrew project that uses the game's 64DD features.

After beating the base game and Master Quest, you would unlock the _Ultimate Trial_ homebrew expansion.

## Majora's Mask
It would use _2ship2harkinian_ and include the best settings and the MM3D 4K texture pack.
Beating the game would unlock the homebrew _Master Quest_ version.

## Master of Time
After beating OoT and MM (both the base and Master Quest versions), you would unlock the _Master of Time_ ROM hack.
