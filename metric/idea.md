# Metric Units ROM Hacks
Quite a few video games don't use metric units, and they would be so much better if they had that option.

## Pilotwings Resort
Use km/h and meters in the North American version. Might also have an option to use km/h for speed and feet for altitude.

## Wii Fit / Wii Fit Plus
> [!NOTE]
> Depending on how save data is stored, this mod might need to change the save data structure in the North American version.

You can _kind of_ change the units by using a copy for a different region, but there are some problems with that. It's inconvenient, and it makes it harder to go back. Also, if you want to use pounds and centimeters, then you're out of luck.

Someone should make a ROM hack that allows each user profile to choose their preferred units for height, weight, and speed.

## Wii Sports Resort
Golf uses yards, but some people just aren't used to yards. Someone should make a ROM hack that makes it display the distance in meters in addition to the distance in yards (extra text would be added below the yards display).

## Police Quest (VGA Remake)
It uses MPH when driving. A good ROM hack would change it to km/h.
